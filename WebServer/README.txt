This WebServer is used to provide http access to webmobinet files and 
to be used to compile textcode to Graphic blocs and to viceversa.

to modify the code of the webserver you have to modify .java files 
in the org/jibble/simplewebserver folder

after modifiying the code you have to compile java classes with the 
command below: javac *.java

then you have to generate the jar file with the command executed in this : 
jar cmf META-INF/MANIFEST.MF WebServer.jar .
